# shadcn-ui-setup: shadcn/ui 初期化

## タスク ID: shadcn-ui-setup
apps/content-reviewer/ で shadcn/ui を初期化し、レイアウト用コンポーネントを追加。

## 作成・変更ファイル
- `apps/content-reviewer/components.json` — shadcn/ui 設定（new-york スタイル、zinc ベースカラー）
- `apps/content-reviewer/src/lib/utils.ts` — cn() ヘルパー（clsx + tailwind-merge）
- `apps/content-reviewer/src/app/globals.css` — CSS 変数（light/dark テーマ、sidebar 変数含む）
- `apps/content-reviewer/src/components/ui/button.tsx` — Button コンポーネント
- `apps/content-reviewer/src/components/ui/sheet.tsx` — Sheet コンポーネント（モバイルサイドバー用）
- `apps/content-reviewer/src/components/ui/separator.tsx` — Separator コンポーネント
- `apps/content-reviewer/src/components/ui/sidebar.tsx` — Sidebar 系コンポーネント一式
- `apps/content-reviewer/src/components/ui/tooltip.tsx` — Tooltip（Sidebar 依存）
- `apps/content-reviewer/src/components/ui/input.tsx` — Input（Sidebar 依存）
- `apps/content-reviewer/src/components/ui/skeleton.tsx` — Skeleton（Sidebar 依存）
- `apps/content-reviewer/src/hooks/use-mobile.tsx` — useIsMobile フック（Sidebar 依存）
- `apps/content-reviewer/package.json` — Radix UI、CVA、lucide-react 等の依存追加

## 設計判断
- `npx shadcn@latest init` + `npx shadcn@latest add` で公式 CLI を使用（Tailwind CSS v4 対応済み）
- Sidebar コンポーネントが tooltip, input, skeleton, use-mobile を依存として要求するため全て追加
- Biome の `useExhaustiveDependencies` ルールに合わせ、React state setter を deps から除外

## 次のタスクへの申し送り
- app-layout-shell タスクで Sidebar 系コンポーネントを使ってレイアウトを構築可能
- globals.css に sidebar 用 CSS 変数が既に定義済み
