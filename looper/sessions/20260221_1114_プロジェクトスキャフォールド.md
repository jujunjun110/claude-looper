# project-scaffold: pnpm workspace + Next.js App Router 初期構築

## タスク ID
project-scaffold

## 作成したファイル
- `package.json` — ルート workspace scripts (lint, typecheck, build, verify)
- `pnpm-workspace.yaml` — apps/* を workspace として定義
- `apps/content-reviewer/package.json` — Next.js 15 + React 19 + Tailwind CSS v4 + Biome
- `apps/content-reviewer/next.config.ts` — 最小構成
- `apps/content-reviewer/tsconfig.json` — bundler moduleResolution, `@/*` パスエイリアス
- `apps/content-reviewer/tailwind.config.ts` — src 配下を content に指定
- `apps/content-reviewer/postcss.config.mjs` — @tailwindcss/postcss プラグイン
- `apps/content-reviewer/biome.json` — タブインデント、JS はシングルクォート、セミコロン必須
- `apps/content-reviewer/src/app/globals.css` — `@import "tailwindcss"` (v4 形式)
- `apps/content-reviewer/src/app/layout.tsx` — RootLayout (lang=ja, metadata 設定)
- `apps/content-reviewer/src/app/page.tsx` — 最小限のホームページ

## 設計判断
- Tailwind CSS v4 は `@import "tailwindcss"` ディレクティブ + `@tailwindcss/postcss` を使用（v3 の `@tailwind` ディレクティブではない）
- Biome の CSS フォーマッタはデフォルトでダブルクォートのため、CSS ファイルはダブルクォートに統一
- packageManager フィールドに pnpm@9.15.4 を指定（Corepack 互換）

## 次のタスクへの申し送り
- shadcn-ui-setup: `@/*` パスエイリアスは設定済み。`lib/utils.ts` に cn() を作成し shadcn init を実行する
- prisma-schema: `apps/content-reviewer/prisma/schema.prisma` を新規作成。prisma devDependency の追加が必要
